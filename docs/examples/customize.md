## 处理和预览

React DnD 允许您在组件的呈现函数中选择可拖动节点和拖动预览节点。

您还可以在加载图像后使用以编程方式创建的图像实例。

[示例](https://04onyvvq80.csb.app/)

## 拖放效果

有些浏览器允许你为可拖动项目指定 "拖拽效果"。在兼容的浏览器中，当你把第一个方框拖到下拉区时，会看到一个 "复制 "图标。

[示例](https://2xvk84jqmn.csb.app/)
